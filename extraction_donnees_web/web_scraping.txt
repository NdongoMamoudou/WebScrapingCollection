Definition de Beautiful

Beautiful Soup est une bibliothèque Python utilisée pour extraire, 
analyser et manipuler des données provenant de documents HTML ou XML. 
Elle est particulièrement utile pour le web scraping, 
où l'objectif est de récupérer et structurer des informations disponibles sur des pages web.
